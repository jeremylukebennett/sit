Sit.

How to Use
'Sit' is a simple meditation timer. Use the 'Duration' and 'Interval' sliders to set the length (in minutes) of your meditation and the frequency of interval alerts throughout. Use the 'Sound' slider to select an alert sound.

If you'd like to track your progress over time you can create an account. Make sure you're logged in before beginning the timer if you'd like for your session to be recorded.
About Sit
'Sit' is a free web-based alternative to downloadable, subscription-based meditation apps. It works on mobile and desktop devices.
