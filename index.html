How to Use...
'Sit' is a simple meditation timer. Use the first and second rows of sliders to set the duration of your meditation and the frequency of interval alerts. The third slider is used to select an alert sound.

If you'd like to track your progress over time you can create an account. Make sure you're logged in before beginning the timer if you'd like for your session to be recorded.
